We're excited to introduce **iqtoolkit.ai** - the only open-source toolkit that seamlessly combines multi-database expertise with AI-powered analytics to transform how you work with databases.

## What is iqtoolkit.ai?

iqtoolkit.ai is a comprehensive platform designed for developers and database administrators who need intelligent, automated solutions for database management across MongoDB, PostgreSQL, and more. By leveraging local AI models through Ollama integration, we bring enterprise-grade intelligence to your database workflows without compromising your data privacy.

## Getting Started

Install iqtoolkit.ai in minutes:

```bash
# Clone the repository
git clone https://github.com/iqtoolkit/iqtoolkit.ai

# Install dependencies
npm install

# Configure your database connections
cp .env.example .env.local

# Start the development server
npm run dev
```
